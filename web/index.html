Cedar Floor Planner
Computational Layout Tool

Concept

Cedar Floor Planner is an interactive design tool that visualizes spatial layouts generated from programmatic inputs. It enables users to explore multiple floor plan variants, compare efficiency, and understand spatial distribution through an intuitive interface. The tool bridges computational layout generation and visual analysis without requiring direct interaction with CAD environments.

Methods and Tools

The system combines Python-based spatial computation with SVG-based web visualization. Room programs are processed through a rectangle packing algorithm to generate layout variants, which are exported as structured JSON. JavaScript renders geometry, metrics, and comparisons, while lightweight CSS structures the interface for fast, interactive exploration of design alternatives.
